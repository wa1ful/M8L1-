
## Команды бота

- `/start` - начать работу с ботом

## Функционал

- Автоматические ответы на 6 частых вопросов (оформление заказа, статус, отмена, повреждённый товар, техподдержка, доставка)
- Кнопки с вопросами для удобной навигации
- Обращение к специалисту с сохранением заявки в базу данных

## База данных

Таблица `requests` хранит все обращения к специалистам:
- id - номер заявки
- user_id - Telegram ID пользователя
- username - логин пользователя
- question - текст проблемы
- created_at - дата обращения
